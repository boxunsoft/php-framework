# php-framework

A simple MVC framework with PHP

## Usage

See demos or Documents

## Install

In composer.json
```composer.json
...
  "require": {
    "imarno/php-framework": "1.*"
  },
  "autoload": {
    "psr-4": {
      "Ala\\": "src/"
    }
  }
...
```

## App Directorys
```dir
src
  |- Application
       |- AppName
           |- Controller
                |- Index.php
           |- view
                |- index.php
           |- Controller.php
           |- Error.php
  |- public
       |- appname
            |- index.php
  
  
```
